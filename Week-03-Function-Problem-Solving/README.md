# Week 03 – Task 06: Real-World System Simulations

This folder contains solutions for Task 06 completed as part of the Agentic AI Internship at Innomatics Research Labs.

## Task Included:

1. Smart Parking Lot Management System  
2. Online Food Delivery Time Estimator  
3. Movie Theatre Seat Occupancy Analyzer  
4. Cloud Server Load Classification System  
5. Smart Classroom Resource Usage Monitor  
6. Online Event Registration Capacity Controller  

The objective of this task was to simulate real-world monitoring and control systems across domains like smart infrastructure, cloud computing, event management, and logistics.

# Week 03 – Task 07: OOPS Based System Design

## Task Included:

1. Smart Light Controller  
2. Employee ID Card System  
3. Mobile Contact Record  
4. Product Price Tag Generator  
5. Movie Rating Display System  
6. Delivery Address Manager  

Each problem focuses on applying Object-Oriented Programming (OOPS) concepts such as:
- Class creation
- Constructors (`__init__`)
- Attributes and methods
- Object instantiation
- Structured system modeling

The objective of this task was to understand how real-world systems can be modeled using classes and objects in Python.
