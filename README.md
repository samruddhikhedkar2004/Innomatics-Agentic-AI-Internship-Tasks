# Innomatics-Agentic-AI-Internship-Tasks

This repository contains weekly tasks and assignments completed as part of the Agentic AI Internship at Innomatics Research Labs.

## Structure
- Week-wise folders containing Jupyter notebooks
- Each notebook includes logic building and problem-solving tasks based on session learnings
- Separate README files are maintained for each week to describe tasks and learnings

## Technologies Used
- Python
- Jupyter Notebook

This repository will be updated weekly as new tasks are assigned.
